# 素材放置說明

請把你的圖片、logo、或其他素材放到這個資料夾。

## 建議的子資料夾結構

```
assets/
├── logo/          ← 品牌 logo
├── images/        ← 內容圖片
├── icons/         ← icon 圖示
└── backgrounds/   ← 背景圖
```

## 支援的格式

- 圖片：`.png`、`.jpg`、`.jpeg`、`.svg`
- 其他：`.pdf`（參考用）

## 命名建議

使用英文命名，避免空格，例如：
- `logo-main.png`
- `chapter1-cover.jpg`
- `icon-star.png`
